Single Wordpress Post from WP-API using just Jquery

Usage: Upload these files to your Wordpress site (in their own folder) and then view via browser. Append the ID of the post you want to view to the url. For example if the postID is 123 the url is: [yourdomain.com/yourfolder]?id=123. Thats it!!!
